# Barth-FQI-heiEditions
TEI P5 prototype edition of § 1.1 of Karl Barth's Fides quaerens intellectum (GA II.13). Four-witness apparatus (A 1931 / B Vorabdruck / A2 1958 / GA), bilingual term register, GND-linked authority files. Encoding follows Šimek 2022's five textual dimensions and binds to heiEDITIONS Concepts via @ana. Portfolio prototype.
